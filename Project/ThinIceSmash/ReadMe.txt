Instructions for openning the project:
The project was made in UE4 v4.18.3

Download that or as close to the same version as possible. This project also uses C++, so make sure to have visual studio or some way to access it.
Next, download the project.
Now right click the uproject, and tell it to generate visual studio files.
Next open the .sln, build the project, then once thats build, open the uproject.

If all goes well, you should now have access to the project in UE4.
